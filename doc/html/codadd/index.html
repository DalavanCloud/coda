codadd

This documentation describes the functionality of the codadd tool which is part of CODA.

General description

The codadd tool is a support tool for the CODA Product Format Definition (.codadef) files. It is able to verify the .codadef contents and allows generation of HTML documentation for the product format definitions.

    codadd
        Read all .codadef Product Format Specifications in the CODA definition
        search path and report any problems that may occur in these files.

    codadd doc <location>
        Generate HTML documentation for the .codadef Product Format
        Specifications in the CODA definition search path.

    codadd -h, --help
        Show help (this text)

    codadd -v, --version
        Print the version number of CODA and exit
